 - "G" - smaž podlahu/zed          OME

 - "T" - přidat travu              OME
 - "K" - přidat kámen OME
 - "M" - přidat modrý krystal OME
 - "C" - přidat červený krystal OME

 - "S" - dolu (funguje i šipka) OG
